Projet Unity de NICOLAY Yohan

=============================

Pour bien comprendre le \_\_Projet Unity\_\_, une approche peut être :



 - Ouvrir \*\*Unity Hub\*\* ;



 - Ajouter un projet sur \*\*Unity Hub\*\* avec la version 6000.0.52f1 de \*\*Unity\*\*.



Pour ajouter le projet sur \*\*Unity Hub\*\*,

il est possible d'ajouter le repository associé au Project \_Des cieux aux abysses\_ de l'Organization orkanak165



OU



il est possible d'ajouter le dossier ajouté dans le répertoire personnel et de l'ajouter dans \*\*Unity Hub\*\*.



Le build du projet est en 4K, 60fps.



Le sujet \_Des cieux aux abysses\_ expliqué comme étant:



\_Création d'une scène ouvrant l'accès à de nouveaux plans d'existence transcendant les limites perceptuelles de la condition humaine.\_

\*\*Geoffrey Gorisse\*\*, 7 octobre 2025.



 nous proposait de transcender l'esprit humain pour se confronter à des plans d'existences.

Il est vrai que je me suis, à travers ce projet, essayé à exploiter le caractère disruptif de l'humain par son calme et sa vivacité.

Cela a été une véritable introspection sur ma vie à un moment des plus difficiles...
